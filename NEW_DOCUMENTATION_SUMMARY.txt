=============================================================================
QUICK START COMMAND
=============================================================================

For users who want to start right now:

1. Get Alchemy API key (free):
   - Go to https://www.alchemy.com/
   - Sign up and create Sepolia app
   - Copy your API key

2. One-line setup:
   cp .env.example .env
   # Edit .env and add your RPC_URL

3. Run:
   make db-setup
   make run

4. Open:
   http://localhost:8080/

That's it! Data is automatically fetched and indexed.
